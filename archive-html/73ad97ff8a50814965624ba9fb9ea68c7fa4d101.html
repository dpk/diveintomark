Even though technically I could compile my own version of a program under Linux, I would have to learn a great deal more about the system, and synchronize my changes with the community or have to repatch my version whenever an update was released.

    I love Linux and run it on several devices, but I write code for the Mac (or the web) because I only have finite amounts of time.

    — Dethe Elza
