Address book

(osx, vcard)

OK, LazyWeb, I need an address book. The Address Book.app that comes with Mac OS X is nice for keeping about 10 addresses. We have 100. (We’re planning a wedding here.) Suggestions for a package that either runs on Mac OS X or can be installed as a web app on a Linux/Apache/MySQL/Perl/Python/PHP server. Shareware, freeware, or open source. Must do the following:

  • Import the addresses we have already entered into Address Book. I can export them as vCards. Or, at the very least, import from a tab-delimited file. I can use Address Book Exporter to create a tab-delimited file.
  • That implies that it must be able to handle all sorts of weird information, like instant messenger accounts, multiple email addresses, and so forth.
  • Some other customizeable fields would be nice too.
  • Group people into various overlapping groups.
  • Sort. Amazingly, Address Book.app has no sort function. Or if it does, I can’t find it.
  • Select, say, all the people in a particular state, or all the people in a particular group, or some combination.
  • Print a list. We’re not going to use it for mailing labels, we just want to print nicely. Or…
  • Create a customizeable HTML report of, say, everyone in the Wedding group and their email address.

Such a thing must exist. I am not the only person with this problem.
