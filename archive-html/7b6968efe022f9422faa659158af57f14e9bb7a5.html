I need a name for these lists

(css, linkdump, muse, openoffice, osx, pdf, rss)
  • Dorothea Salo: Why PDF sucks, which leads Jonathon Delacour to discover OpenOffice, which truly does rock, except on Mac OS X (this beta release can save files), where it not so much rocks as rolls quietly and inexorably towards total world domination.
  • imageJaguar.sh is a shell script to automate creating and restoring Apple Software Restore and NetBoot images. If you are one of the few people in the world for which that last sentence means anything, you are probably leaping out of your chair for joy now. Otherwise, probably not.
  • Apple: ASCIIMoviePlayerSample demonstrates using a MovieDrawingCompleteProc to render a QuickTime movie in the Mac OS X Terminal. Yes, it’s true, now you can enjoy the latest QuickTime Movie Trailers without leaving the comfort of the command line.
  • Ian Hickson: I’ve been writing lots of web standards test cases recently, including a bunch of CSS tests.
  • Michael Barrish: Garbage. Many years ago I tripped on mushrooms with a friend in the low hills outside Santa Fe. We wandered through a maze of prickly bushes and speculated about the people who had lived there once. Had any walked where we now walked? Had any seen what we now saw? What thoughts had they had, thoughts unthinkable to us, two men from a world which bore little relation to theirs, though it occupied the same space. While pondering these things we came upon an empty Doritos bag.
  • Phil Ringnalda: Now that’s ironic. The RSS 2.0 item for Sam’s post is perfectly valid, correct, best practice RSS: the description element contains plain text, with the characters that XML requires be escaped escaped. The HTML for the item is in an entity-encoded content:encoded element. It’s just not possible to do better RSS than that, and it took me three aggregators and two browsers to read the item.
  • Brian Jespen: Top Ten Mac OS X Tips for Unix Geeks. /etc is Not Always in Charge
  • Scott Andrew LePera: These sites do great CSS
  • Margaret Berry: Women’s Fashion: Part 1, Classics. Mom was right. That dress did make you look like a hussy.
  • Firda Beka: Copy Pretender. I have mastered the art of looking busy, but now that I’m self-employed, that skill has become pretty much useless. What a waste.
  • Mac OS X Hints: Use freedb.org instead of CDDB.
  • Mac OS X Hints: Autostart Internet Sharing.
  • Simon Willison: XML-RPC interface for W3C HTML Validator.
