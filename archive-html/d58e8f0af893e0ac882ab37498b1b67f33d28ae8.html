Things that are awesome

(fonts, goats, java, linux, software, wordle)
  • FontMatrix, which helped me pick the God-awful fonts I’m foisting on the world
  • PowerTOP, because I love viewing new cross-sections of my computing environment
  • Wordle (example), despite being a Java applet written in 2008
  • OpenJDK, which is now in Debian main
  • YSlow, which proved that my Cache-Control headers really did blow goats
  • WP-SuperCache, which fixed them (sorry, goats)
  • Flight of the Conchords (note: they are actually twice as awesome as everything else on this list)
  • Trader Joe’s, because food wasn’t expensive enough already
  • Toblerone, just because

  2. Firefox 3, whose “Reopen tabs from last session” functionality allowed me to recreate 2 hours of prep work in 10 minutes after my system locked up. Now I know what I’ll be counting on for that sort of thing.

    Gmail’s IMAP implementation, which remaps “Delete” as “Remove from Inbox/Label” except in Spam/Trash where it actually deletes. Seems incongruous but it makes a lot of sense to me (Gmail’s philosophy being that “archive” is the new “delete”). Moving a message to a label “folder” just adds that label to the message, and there is a special “Starred” folder.
