  3. sudo aptitude install imagemagick

    In a directory of any-shaped originals (highly destructive, make a backup first!):

    mogrify -verbose -resize x360 -resize '480x<' -resize 50% -gravity center -crop 240x180+0+0 +repage *

    And to create the poster:

    montage * -geometry 240x180x0x0 -tile 18x36 ../output-4320x6480.jpg

    or

    montage * -geometry 240x180x0x0 -tile 27x24 ../output-6480x4320.jpg

    or

    montage * -resize 120x90 -geometry 120x90x0x0 -tile 6x108 -quality 50 ../output-720x9720.jpg

    I did whip up a stupid Python script to randomly rename the pictures so I could get different posters. montage doesn't have a randomize flag.
